How to Restart SAP Business One Service On SUSE Linux Server

Follow there steps about how to restart SAP HANA related services in SuSE Linux Enterprise Operating System.

1- Hold any transaction from all users, logout all user.

2- Open puTTy, and logon to your SAP HANA System using user root, then change user to <SID>adm..

3- Change directory to HANA installation directory <HANADIR>/<SID>/HDB00.

4- Type command below to stop HANA Database Engine Service and hit enter.

 

5- Wait until service stopped, then type command below to start HANA Database Engine Service

 

6- Wait until service fully started.

 

7- Return to user root by typing exit.

8- Type command /etc/init.d/sapb1servertools restart to restart SLD service, Analytics Platform, etc.

 

9- Wait until service fully restarted.

 

10- Type command /etc/init.d/b1s restart to restart service layer services.
